# Client Scheduler

A robust, web-based scheduling application designed for efficient appointment and task management. This project provides a clean, user-friendly interface for tracking schedules and maintaining personal or professional tasks.

## Project Overview
The Client Scheduler is a productivity tool built to help users manage clients time and organize their appointments effectively. The application emphasizes simplicity, reliability, and seamless user access.

## Key Features
- **User Authentication:** Secure login integration via Supabase, including Google OAuth support.
- **Task Management:** A dedicated interface for creating, viewing, and organizing scheduled appointments.
- **Responsive UI:** Designed to work across various screen sizes, ensuring accessibility on both desktop and mobile devices.
- **Technology Stack:**
  - **Frontend:** React with Vite.
  - **Backend/Database:** Supabase (Auth, Database).
  - **Deployment:** Vercel.

## Getting Started

### Prerequisites
- Node.js (v18+)
- A Supabase project account
- Vercel account for deployment
